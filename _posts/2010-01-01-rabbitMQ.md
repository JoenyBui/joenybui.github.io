# RabbitMQ

* open-source message-broker 
* implemnts Advanced Message Queuing Protocol (AMQP)
* data delivery, non-blocking operations or push notifications
* public / subscribe
* asynchronous processing
* work queues
* reliability
* flexible routing
* clustering
* federation
* highly available queues
* multi-protocol
* many clients
* management UI
